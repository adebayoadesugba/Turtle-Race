# 🐢 Turtle Race Game (Python Turtle)

A fun **Turtle Racing Game** built using Python and the Turtle graphics module.  
In this game, the player places a bet on which turtle will win the race. Multiple turtles race across the screen with random speeds, and the first turtle to cross the finish line wins.

This project demonstrates the use of **Python loops, lists, randomization, user input, and the Turtle graphics module** to create a simple interactive simulation game.

## Features

- Interactive betting system
- Multiple turtles racing simultaneously
- Random movement speed for each turtle
- Real-time race animation
- Automatic winner detection
- Displays race result and player outcome

## Requirements

Before running the game, make sure you have **Python 3.x installed**.

You can download Python from:

https://www.python.org/downloads/

To check if Python is installed:

### Windows

```bash
python --version
```

### MacBook / Linux

```bash
python3 --version
```

## How to Run the Game

Follow these steps to run the game on your computer.

### 1. Clone the Repository

```bash
git clone https://github.com/adebayoadesugba/Turtle_Race_Game.git
```

### 2️. Navigate into the Project Folder

```bash
cd turtle-race-game
```

### 3️. Run the Game

#### Windows

```bash
python main.py
```

#### MacBook / Linux

```bash
python3 main.py
```

The game window will open and ask you to **place a bet on which turtle will win the race**.

---

## Game Rules

- You will be asked to bet on a turtle color
- Five turtles will start racing from the left side of the screen
- Each turtle moves forward at a random speed
- The first turtle to reach the finish line wins
- If your bet matches the winning turtle, you win
